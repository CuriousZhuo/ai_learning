## Python 基础数据类型

| 类型 | 例子 | 一句话 |
|------|------|--------|
| `str` 字符串 | `"hello"`, `"photo.jpg"` | 一串文字，能做判断（开头/结尾/包含） |
| `int` 整数 | `1`, `100` | 数字，能做加减乘除 |
| `float` 小数 | `3.14` | 带小数点的数字 |
| `bool` 布尔 | `True`, `False` | 对/错，是/否，判断的结果 |
| `list` 列表 | `["a", "b", "c"]` | 一堆东西排好顺序，能遍历 |
| `tuple` 元组 | `("name", ".jpg")` | 固定组合，不能改，能拆包赋值 |

### 重要区分
- `.` 在 `"path/./file"` 里是"当前目录"
- `.` 在 `os.listdir` 里是"os 工具箱里的 listdir 工具"
- 两个句号，完全不同含义

## 学到的 Python 语法

### import
```python
import os  # 把操作系统工具箱拿出来
```
内置的（print、len）不需要 import，操作系统相关的需要 os，数学相关的需要 math。

### 变量
```python
folder_path = "./test_files"  # 贴标签，把值装进去
```

### for 循环
```python
for name in all_files:  # 对列表里每一个元素，执行下面缩进的部分
    print(name)
```

### if 判断
```python
if name.endswith(".jpg"):  # 如果返回 True 就执行下面
    print(name)
# 没有 else = 不满足就跳过
```

### f-string 格式化字符串
```python
name = "photo_001"
f"IMG_{name}"  # → "IMG_photo_001"
```
`{}` 里的变量会被替换成它的值。

### 元组拆包
```python
name_part, ext_part = os.path.splitext("photo_001.jpg")
# name_part = "photo_001", ext_part = ".jpg"
```
右边返回两个值，左边两个变量一一对应装入。

## 学到的 os 模块函数

| 函数 | 作用 | 输入 | 输出 |
|------|------|------|------|
| `os.listdir(path)` | 列出文件夹下所有文件名 | 文件夹路径字符串 | 文件名列表 |
| `os.path.splitext(name)` | 切开文件名和后缀 | 文件名 | (名字, 后缀) 元组 |
| `os.path.join(dir, name)` | 拼接完整路径 | 文件夹, 文件名 | 完整路径字符串 |
| `os.rename(old, new)` | 重命名文件 | 旧路径, 新路径 | 无 |

### 字符串方法
| 方法 | 作用 | 例子 | 结果 |
|------|------|------|------|
| `.endswith(suffix)` | 判断是否以某后缀结尾 | `"a.jpg".endswith(".jpg")` | `True` |
| `.startswith(prefix)` | 判断是否以某前缀开头 | `"IMG_a".startswith("IMG_")` | `True` |

## 关键心得

1. **数据结构不是什么高深的东西**——就是数据能长什么样（str/int/bool/list/tuple）和能对它们做什么操作
2. **编程就是拼积木**——每个函数只做一件小事，把它们串起来就变成完整功能
3. **先学概念，让 AI 写代码，自己审查**——这个模式效率很高，不卡在语法细节上
